# getx_boilerplate

A minimal Flutter + GetX boilerplate with:
- Splash, Home, Auth(Login,Register) modules (GetX bindings/controllers/views)
- Localization (English default, Arabic)
- Light/Dark themes
- ApiService (no default base URL)
- AppImages, AppIcons, ApiUrls utility files

## How to use:
1. Clone this repository Or Unzip project.
2. Run `flutter pub get`.
3. Run `flutter create .`
4. Run the app: `flutter run`.

Included translation files: assets/lang/en.json, assets/lang/ar.json
